# Sudoku Engine

A cross-platform sudoku solver/generator that compiles to both native executable and WebAssembly.

## Project Structure

```
sudoku-engine/
├── CMakeLists.txt
├── include/
│   └── sudoku.h
├── src/
│   ├── main.cpp
│   └── sudoku.cpp
├── frontend/           (your Vue.js project)
│   └── public/
│       └── wasm/       (auto-generated by build)
└── build/              (created during build)
```

## Building for Native

```bash
# Create build directory
mkdir build && cd build

# Configure
cmake ..

# Build
cmake --build .

# Run
./sudoku-engine
```

For Release build:
```bash
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```

## Building for WebAssembly

Requires [Emscripten SDK](https://emscripten.org/docs/getting_started/downloads.html) to be installed and **activated**.

```bash
# Create build directory
mkdir build-wasm && cd build-wasm

# Configure with Emscripten
emcmake cmake ..

# Build (this will automatically copy artifacts to frontend/public/wasm/)
cmake --build .
```

The WASM build outputs an ES6 module that can be imported in the web frontend:

```javascript
import createSudokuModule from "/wasm/sudoku-engine.js"

const Module = await createSudokuModule()
// Use your sudoku engine here
```

The build will automatically create `frontend/public/wasm/` and copy:
- `sudoku-engine.js` - ES6 module wrapper
- `sudoku-engine.wasm` - WebAssembly binary

## Requirements

- CMake 3.15+
- C++ 20 compatible compiler
- Emscripten SDK (for WebAssembly builds)
